# 2.3 光纤的优点及其两种模式

（P50 2.3.1）光纤由石英玻璃拉成细丝制成，纤芯直径只有 $8$ ~ $100$ $\mu m$。

## 2.3.1 光纤的优点（P52 2.3.1）

+ 传输损耗小，中继距离长。
+ 抗雷电和电磁干扰性能好。
+ 无传音干扰，保密性好。
+ 体积小，重量请。

## 2.3.2 光纤的两种模式（P51 2.3.1）

+ **单模光纤 SMF（Single-Mode Fiber）**

  单模光纤支持单一光传输模式，适合长距离、高带宽应用，但成本较高。

+ **多模光纤 MMF（Multi-Mode Fiber）**

  多模光纤支持多重光传播模式，适合短距离传输。

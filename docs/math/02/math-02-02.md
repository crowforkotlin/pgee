# 5. 特征值和相似

## 特征值

若有 $n$ 阶矩阵 $A$，存在特征值 $\lambda$ 和特征向量 $ξ$，则：

+ $A ξ=\lambda ξ$
+ $(kA) ξ=(k\lambda) ξ$
+ $(A^k) ξ=(\lambda^k) ξ （若 \; A \; 可逆，则 \; k \; 允许负数）$
+ $f(A) ξ=f(\lambda) ξ$
+ $(A^*) ξ=(\frac{|A|}{\lambda}) ξ$
+ $(P^{-1}AP) ξ=\lambda(P^{-1} ξ)$
+ 若 $f(A)=O$，则 $f(\lambda)=0$

## 相似

若 $A \sim B$，则有：

+ $r(A)=r(B)$
+ $|A|=|B|$
+ $tr(A)=tr(B)$
+ $对于任意的 \; \lambda，有 \; \lambda E-A \sim \lambda E-B$
+ $\lambda_A=\lambda_B，即\;A、B\;具有相同的特征值$

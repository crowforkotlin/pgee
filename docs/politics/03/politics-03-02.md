# 基本问题

## 41. 全面深化改革

深化改革总目标：

+ （**根本方向**）完善和发展中国特设社会主义制度
+ 推进国家治理体系和治理能力现代化

深化改革目标是：

+ 同十八届三中全会一脉相承
+ 同二十大相衔接
+ 为进一步全面深化改革明确了目标导向

全面深化改革：

+ 主题：推进中国式现代化
+ 核心问题：处理好政府和市场的关系
+ 重点：以经济体制改革为**主线**
+ 出发点和落脚点：（人民）以促进社会主义、增进人民福祉

二十届三中全会：

+ 通过《中共中央关于进一步全面深化改革、推进中国式现代化的决定》
  + 坚持党的全面领导
  + 坚持以人民为中心
  + 坚持守正创新
  + 坚持以制度建设为主线
  + 坚持全面依法治国
  + 坚持系统理念
